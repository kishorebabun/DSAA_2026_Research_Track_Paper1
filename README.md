# DSAA_2026_Research_Track_Paper1
DSAA_2026_Research_Track



Code Availability and Reproducibility

This repository contains the complete implementation of the proposed federated learning framework presented in the DSAA 2026 Research Track submission. The codebase includes all components required to reproduce the experimental results reported in the paper, including data preprocessing, model training, federated aggregation, privacy-preserving mechanisms, adaptive client selection, and performance evaluation.

The proposed framework is evaluated on three benchmark datasets representing diverse data characteristics and application domains:

Medical Dataset
MNIST Dataset
CIFAR-10 Dataset

The repository provides:

End-to-end implementation of the proposed federated learning methodology.
Training and evaluation pipelines for all experimental settings.
Data preprocessing and partitioning procedures used in the federated environment.
Differential privacy integration and communication-efficient optimization mechanisms.
Experimental scripts for reproducing accuracy, convergence, privacy, and communication-efficiency results.
Comparative evaluation against baseline federated learning approaches.

The implementation is organized to facilitate reproducibility, allowing researchers and practitioners to replicate the reported findings, validate performance under different configurations, and extend the proposed framework for future research.

The repository serves as the official implementation accompanying the DSAA 2026 submission and is intended to support transparency, reproducibility, and further advancements in privacy-preserving federated learning.
